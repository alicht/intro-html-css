### lets create an index.html file

here's the boilerplate HTML code we'll want to enter in our HTML file:

```html
<!DOCTYPE html>
<html lang="en">
  
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" type="css" href="style.css" />
    <title>Title Goes Here</title>
  </head>
  
  <body>
  
  </body>
  
</html>
```
